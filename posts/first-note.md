# 第一篇笔记

这是我的第一篇博客笔记，记录博客的创建过程。

## 为什么选择 VitePress？

VitePress 是一个基于 Vite 的静态站点生成器，具有以下优势：

- ⚡ **极快的构建速度**：基于 Vite，开发体验极佳
- 📝 **Markdown 优先**：专注于内容创作
- 🎨 **主题丰富**：默认主题美观且可定制
- 🔍 **内置搜索**：支持本地搜索功能
- 📱 **响应式设计**：完美支持移动端

## 开始使用

创建这个博客非常简单：

1. 使用 `npm create vitepress@latest` 初始化项目
2. 在 `posts/` 目录下创建 Markdown 文件
3. 运行 `npm run docs:dev` 开始写作
